## Sections

- **Hero** — Introduction with floating identity tags
- **About** — Who I am and what I believe in
- **Education** — Timeline card (Parul University, B.Tech CSE × Microsoft)
- **Technical Skills** — Languages, Concepts, Web Dev
- **Projects** — FinInsight, TCS Clone
- **More About Me** — Strengths, Languages, Interests
- **Contact** — Location, Email, Phone

## Built with

- HTML5
- CSS3 (Flexbox, Grid, CSS animations, transforms)

- Color palette: Forest green (#2d4a2d) + Warm gold + Cream
- Scrolling marquee ticker — pure CSS
- Alternating dark/light sections
- Fully responsive layout
